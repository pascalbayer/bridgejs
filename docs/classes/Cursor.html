Cursor Class

Defined in: server/lib/lib.bridge.js:299

USAGE: Anywhere
The Cursor class is not directly accessible but it is returned whenever you call the .find() method of a collection. A Cursor object provides standard methods to filter your result set.

Limit the result set

Parameters:

  • limitValue Integer

Returns:

Cursor:

Example:

Limit your results. Provide the number of results you want to retrieve. You will get back the first results found in the database within the limit. Normally the results are stored in the database based on the order they where inserted in.

 var Posts = new Bridge.Collection('Posts');
 Posts.find().limit(5);

Create a complete result set

Parameters:

Returns:

Cursor:

Example:

Create an array of all results matching the current cursor options. Warning, the array is completely mapped on the servers main memory depending on the number of items within the collection that could cause a lot of memory usage. If you do not need to use the toArray function, you could use stream instead to save memory

 var Posts = new Bridge.Collection('Posts');
 Posts.find().toArray(function (items) {
     console.log(items);
 });
